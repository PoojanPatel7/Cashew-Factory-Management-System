# CashewPro ERP — User Guide

Welcome to CashewPro! This guide helps you navigate the system based on your role.

## 1. Owner & Manager
*   **Dashboard:** Shows total revenue, P&L, today's output, and factory alerts.
*   **Procurement:** Approve purchase orders, monitor supplier payments.
*   **Sales:** Approve large sales orders and monitor dispatch status.
*   **Reports:** Access all 30+ reports via the `/reports` module and export to Excel/PDF.
*   **Settings:** Manage Users, Security, Backup, and Data Export.

## 2. Supervisor
*   **Dashboard:** Shows active processing lots, team attendance, and floor status.
*   **Processing:** Track raw cashew nuts moving across the 12 stages (Cleaning, Steam, Shelling, etc.). Log output and wastage.
*   **HR:** View employee lists, log piece-rate work (e.g., kg shelled per worker), and manage attendance.

## 3. Accountant
*   **Dashboard:** Shows cash balance, pending payables, receivables, and quick actions for logging expenses.
*   **Accounting:** Manage the General Ledger, create Expense entries, process Payroll distributions, and handle Bank Accounts.

## 4. Operator
*   **Dashboard:** Displays the live status (temperature, pressure, power) of assigned machinery.
*   **Machinery:** Log daily machine work hours and report any breakdowns directly to the supervisor.

## 5. Worker
*   **Dashboard:** A personal portal to view daily attendance status and total piece-rate earnings for the month.
*   **HR:** Option to self-check-in using the app (if enabled by factory policies).
